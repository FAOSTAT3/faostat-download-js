A noter

Les utilisateurs sont priès de prendre note qu'une nouvelle mèthodologie d'imputation de donnèes non-dèclarèes ou mancantes a ètè adoptèe en 2011. Cette mèthodologie a ètè appliquèe aux donnèes à partir de 1990 et les donnèes imputèes en suivant cette methodologie sont indiquèes par le symbole 'Im'.

Les agrégats sont la somme des données disponibles. Pour certains agrégats d'items, des facteurs de conversion sont appliqués aux valeurs lors du calcul des totaux. Veuillez consulter les métadonnées pour connaître ces facteurs.
