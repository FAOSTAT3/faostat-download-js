Trade Indices are for Crops and Livestock, Primary and Processed.

Units

Values for Import Value at Base Period Quantity, Export Value at Base Period Quantity, Import Value at Base Period Price and Export Value at Base Period Price are given in 1000$. Index numbers are a figure reflecting a change in value or quantity as compared with a standard or base. The base usually equals 100 and the index number is usually expressed as a percentage.
