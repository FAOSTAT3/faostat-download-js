Le domaine de Ressources présente les facteurs de production pour le secteur agricole. Au sens large, cette section illustre en détails comment les pays diffèrent dans leur dotation en trois types classiques de ressources: main d'oeuvre, terres et capital. Les différences qualitatives sont importantes pour chacun de ces facteurs de production, mais sont extrêmement difficiles à synthétiser dans un seul indicateur pour les terres, dont la productivité dépend considérablement de l'eau et des caractéristiques des sols.
